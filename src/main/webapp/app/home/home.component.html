It consists in:

  • An Angular front-end with Bootstrap, HTML 5 and SASS
  • A Spring Boot back-end exposing a fictitious PENSIONS domain, with all its required business objects, entities, services, DTO, converters, mappers and the other mandatory components.
  • A JWT based authentication and authorization security.
  • Front-end unit, integration and E2E tests with Jest and Protractor.
  • Back-end unit, integration and E2E tests with Junit, Gatling and Cucumber.
  • H2 and PostgreSQL databases.
  • Caching with Infinispan
  • JPA persistence with Spring Data and Hibernate.
  • Data-model migration management with LiquidBase.
  • Use of SonarQube for the code quality management
  • Use of Docker to simplify the deployment
  • CI/CD
